Шаги для настройки бота - GROQ:
1. Перейдите на сайт: https://console.groq.com/ "Если вы из России включите VPN - Если сайт не работает".
2. Нажмите Login и войдите, но в нашем случае зарегистрируйтесь.
3. Дальше найдете Create API Key. 
4. Введите название и нажмите создать.
5. Скопируйте полученный API Key и вставьте в код. 
Шаги для настройки бота - Создание ТГ бота:
1. Скачайте Telegram (Если ещё не установили)
2. Введите в поиск @botfather в Telegram.
3. Нажмите СТАРТ. 
4. Напишите команду боту: /newbot
5. Введите название бота, после чего псевдоним.
6. Готово! Вы получили ваш долгожданный токен! Осталось вам его вставить в ваш код!

Все шаги окончены.
